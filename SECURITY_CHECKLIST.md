# Checklist de seguridad para producción

- Cambiar SECRET_KEY
- Cambiar contraseñas demo
- Usar PostgreSQL, no SQLite
- Activar HTTPS
- Limitar tamaño de archivos
- Respaldos automáticos de base de datos
- Revisar permisos de uploads
- Configurar logs del servidor
- Definir política de contraseñas
- Restringir acceso al panel admin
